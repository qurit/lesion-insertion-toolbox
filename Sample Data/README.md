# Sample data

PET and SPECT example datasets are hosted on Zenodo together (not on GitHub).

Download link: see the **Sample data** section in the main [README](../README.md#sample-data).

## After download

Unzip into this folder so you have:

```text
Sample Data/
├── PET/
│   └── LIST/
│       ├── LIST*.BLF
│       └── corrections_*.h5
└── SPECT/
    ├── CT/
    └── Projection/
```

Load from the **Setup** tab in the application.

The lesion library (`LesionLibrary.json`) is in the repo root, not in this folder.
